
# SwapCreate dedi

Best swap Creator for dedi in cagayan de ohio




## Installation

Execute This to start Creating swap on your dedi. ( must be in root user )

```bash
bash <(curl -s https://raw.githubusercontent.com/ewn69/swapcreate-dedi/main/create.sh)
```

## Uninstalling the swap file.
    
Execute This to start Removing the swap on your dedi. ( must be in root user )

```bash
bash <(curl -s https://raw.githubusercontent.com/ewn69/swapcreate-dedi/main/remove.sh)
```
